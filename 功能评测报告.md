## 1. 项目概述

FinDataHub 是一个基于 Web 的上市公司财务数据管理平台，支持多维度财务数据的增删改查、Excel 导入导出、可视化分析等功能。

### 技术栈
- **后端**: Python 3.9+, Flask 3.0, SQLAlchemy, MySQL 8.0
- **前端**: Vue 3, Element Plus, ECharts, Vite
- **数据处理**: Pandas, OpenPyXL
